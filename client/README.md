# Portfolio Client

Frontend React + TypeScript pour l'application portfolio.

## Technologies

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- React Router
- Axios

## Installation

```bash
npm install
```

## Développement

```bash
npm run dev
```

Le serveur de développement démarre sur http://localhost:5173

## Build

```bash
npm run build
```
